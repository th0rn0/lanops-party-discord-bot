# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2026-04-14

### Added
- Initial release of lanops-party-discord-bot
- Discord bot with prefix-based command routing
- Jukebox commands: current, pause, skip, start, stop, volume
- Stream commands: enable, list
- Admin role gating for privileged commands
- Drone CI pipeline with lint, test, Docker build/push, and GitHub release steps
